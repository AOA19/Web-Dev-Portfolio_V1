Project Purpose and Goal

This project was completed as a Frontend Mentor challenge, and built with HTML, CSS, and Vanilla JavaScript. The purpose of this challenge was to recreate this landing page, and get it as close to the provided JPG design file as possible.

I chose this project because I wanted to learn and practice making more responsive layouts. And this specific project tested my ability to use not only Flexbox but practice using CSS grid to create dynamic layouts seen throughout the landing page.

Problems and Thought Process

The biggest problem I ran into while working on this project, was building a functional mobile navigation bar. After researching mobile navigation bars, I decided to use JavaScript to create the functionality in my mobile navigation bar. I chose this option because I wanted to continue learning and practicing JavaScript.

Once I was able to style the mobile navigation bar. I needed to write the necessary JavaScript code that would toggle and close the navigation bar when the hamburger menu was clicked by the user. I created a list similar to the one below that broke down each item needed to build out the functionality of the hamburger menu.

  • Write a line of code that will toggle the navigation menu when the hamburger icon was clicked..
  • Once the hamburger icon was clicked replace the icon with an "x" icon so the user can close the menu.
  • Write a line of code that will close the navigation menu when the "x" icon is clicked
  • Write a line of code that will close the navigation menu when the user clicks on one of the navigation links.

Doing this process helped me understand what the code was doing. When I would write one line of code I would go back to the project and see the changes. I was able to make sure I understood what that specific line of code was doing before I moved to the next task.

Lessons Learned

Though this project had its challenging parts. I did learn a lot about CSS grid, and how useful it is when making responsive layouts. Now for my future projects, I try to use CSS grid and flexbox as much as possible. Not only did I learn how to create a mobile navigation bar in JavaScript, but I also learned how important it is to add web accessibility into my JavaScript code. Web accessibility is something I continue to learn, and challenge myself to add into any project I work on.
