Project Purpose and Goal

The BIPOC Wellness Directory was built with HTML, CSS, and React for the GDI (Girl Develop It) 2022 #hackforhealth Hackathon. I collaborated with a team of; 8 UI/UX designers and developers to create a mockup application of a wellness directory that provides BIPOC healthcare providers for people of color.

As members of the BIPOC community, we felt that there was a need for easier access to healthcare providers with whom we identify. We believe that representation matters, and how people are treated within the medical system will influence whether they seek help in the future. The app will allow users to search by type of provider, specific health concerns, location, languages, and insurance accepted. In addition, verified providers will be able to create profiles specifying the type of services they offer.

My Work on the Project

  • I designed and built the login and sign-up forms to enable both users and providers to create accounts for the directory.
  • I created the profile design for the healthcare providers that lists their ratings, specialties, payment options, and languages spoken.
  • I built the functionality of the navigation bar to ensure users can navigate seamlessly through the website, whether they are accessing it from a desktop computer, tablet, or mobile device.

Lessons Learned

By working on this application with a multi-disciplinary team, I gained valuable experience in effectively communicating my designs and code to my teammates. Because the hackathon spanned four days, I had to quickly learn how to develop an application in a short amount of time. While we weren't able to accomplish everything we wanted during the hackathon, we hope to continue working on this application in the future and turn it into a real-life project.
