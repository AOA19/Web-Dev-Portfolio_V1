Project Purpose and Goal

This project was completed as a FreeCodeCamp certification project, built with HTML, CSS, and Vanilla JavaScript. This project aimed to build a landing page that markets a product of my choice. I was able to incorporate the knowledge I learned from the FreeCodeCamp tutorials to build this landing page.

I wanted to take this project a step further by incorporating a shopping cart feature built with JavaScript. The user will be able to add items to their shopping cart, increase the quantity of the items, remove items from the cart, and see their total amount. I initially built this landing page without the shopping cart, but I wanted to add a complex element to the page. This was a good opportunity for me to use JavaScript to create a feature commonly used on product websites.

Problems and Thought Process

While working on the shopping cart functionality, I ran into a couple of issues along the way. For example, removing added items from the cart when the remove button was clicked. When I initially wrote the function intended to remove one item at a time from the cart, everything inside the cart would disappear, which caused an error to occur. I went though my code to see what I was missing. I realized that the function I wrote was trying to access multiple remove buttons for each item, but there is technically only one remove button per item row. So from there I accessed the first index, and that was able to solve the problem.

Another difficulty I dealt with while building the project was designing the look of the landing page. Unlike other projects I've completed, I wasn't given a design file that I would replicate in code. With this project, I was given specific HTML elements that needed to be in my code, but the design of the page was up to me. I chose to use Figma to create a mockup of my design before I started coding it. Creating a design from scratch was challenging because I went back and forth about what the overall layout should look like. My experience using Figma to create my design was rewarding because it helped me improve my design abilities.

Lessons Learned

I learned that implementing a design into code, whether it's someone else's design or my own, is the easiest; and preferred way for me to code. Having a design file allows me to practice building pixel-perfect sites. I learned a lot about JavaScript methods as well, and I used various ones throughout my code. Some methods I was familiar with, while others were new to me. JavaScript's methods were very important to the functionality of the shopping cart. Each one that I used was unfamiliar to me, I made sure to do thorough research on it, so I had a better understanding of what the specific method was doing in my code.
