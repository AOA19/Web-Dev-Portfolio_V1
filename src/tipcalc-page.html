Project Purpose and Goal

This project was completed as a Frontend Mentor challenge and built with HTML, CSS, and Vanilla JavaScript. As part of this challenge, I built a tip calculator that displays the tip amount and total amount per person after a user enters their bill amount, selects the tip percentage, and enters the number of people. In choosing this project, I wanted to test my knowledge of JavaScript operators, functions, and statements, as well as to practice implementing accessibility.

Problems and Thought Process

This project had its challenges. One difficulty; I encountered was crafting the code that would accurately calculate the tip per person amount. The issue with the tip calculation ultimately affected the total bill per person calculation. When the tip button was clicked or a custom tip was entered, it would result in an undefined value for the tip per person. I realized the undefined value occurred because I separated the for loop, a statement that would loop through each tip button to get the value, from the function that would calculate the bill. By placing the for loop statement inside the function, I was able to resolve the issue.

Another issue I faced was maintaining the accessiblity of the tip buttons. I decided to use <input type="radio"> for my tip buttons. To match the design of the buttons I initially set the display property of the radio inputs to none. But doing this eliminated keyboard focusability on those tip buttons. To fix this problem, I removed the display property replaced it with opacity:0, and added a border to the tip buttons whenever they are focused on. In making this change, users who navigate the web using their keyboards will be able to tab through each of the tip buttons.

Lessons Learned

The experience I gained from this project has given me a greater understanding of using numerous statements (e.g., iteration and conditional) inside a function to perform specific tasks. Having my loop statements not work as intended encouraged me to investigate why. My thorough research and testing enabled me to find solutions that ultimately resulted in a working calculator.
