About Me

Hi I'm Alexandra!

I’m a software engineer, passionate about integrating code and design to create dynamic web applications. Throughout my academic and professional career, I've embraced new challenges, from choosing a major and career in electrical engineering to eventually following my creative pull toward software engineering. I was first exposed to software engineering in college, and while it sparked my interest, I was already committed to my electrical engineering studies. As an electrical engineer, I took on the challenge of learning unfamiliar software by asking questions when I was stuck and allowing myself to fail a few times until I mastered it. I’ve applied the same approach to software engineering. In my journey as a software engineer, I tackled new skills that were challenging to learn initially but gradually developed the expertise needed to succeed.

When I'm not coding, you can find me testing out a new recipe to bake, staying active by playing soccer and tennis, and killing it on the karaoke mic. I enjoy creating things from conception to production; whether it’s a web application or a new recipe, I’m always up for the challenge! If you're interested in working together, feel free to get in contact with me below.

Skyfall

Skyfall is a collaborative project developed for Chingu Voyage 45. This app enables users to search and filter meteorite data from the NASA Meteorite Landings API by name, composition, mass range, and year of strike. Users can view the filtered data in a table, explore the geolocation of the strike, and visualize the data through various charts.
