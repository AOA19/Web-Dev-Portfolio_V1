Project Purpose and Goal

This project was completed as a Frontend Mentor challenge, built with HTML, CSS, and Vanilla JavaScript. The purpose of this challenge was to recreate this coming soon page and get it as close to the provided JPG design file as possible.

The goal of this challenge was to practice form validation, by creating an email form that would display a custom error message if the input field is left empty or the user entered an incorrectly formatted email address.

Problems and Thought Process

This was one of my earlier projects that I completed, so I dealt with a few challenges. The first challenge I faced was making the hero image responsive. To solve this issue, I used the HTML picture element to display various image sizes for different screens. Doing so allowed for only a large image to load on larger displays, and a small image to load on smaller displays. Which saves bandwidth, and speeds up the time it takes for the page to load.

Another challenge I dealt with was implementing the necessary code to overwrite the browser form validation with a custom form validation. I needed to change the color of the input field border, add an error icon, and display a custom message whenever the form was submitted incorrectly. But before that, I needed to figure out how to validate the inputted email address.

To solve this, I added the onsubmit event attribute inside the form tag, so when the form is submitted the JavaScript code will be executed. For the JavaScript code, I used a regular expression literal with an if else statement. When the user enters their email address, the code will run and check whether the email matches the character combination of the regular expression before it iterates through the rest of the code. Once I was able to determine how to validate the email address, the styling of the input, and the custom message were simple after that.

Lessons Learned

Before this project, I had not used JavaScript a lot in practice, but this challenge helped me practice basic form validation, and understand JavaScript syntax. I was also able to learn about JavaScript regular expressions; which is important for any kind of form validation. After completing this challenge, I felt a lot more confident going into other projects that required JavaScript.
